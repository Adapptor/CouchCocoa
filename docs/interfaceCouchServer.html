- (CouchDatabase*) databaseNamed: (NSString *)  name

Just creates a CouchDatabase object; makes no calls to the server.

The database doesn't need to exist (you can call -create on it afterwards to create it.) Multiple calls with the same name will return the same CouchDatabase instance.
